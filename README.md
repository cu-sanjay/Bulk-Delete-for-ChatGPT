# **ChatGPT Bulk Chats Delete Tool**

## **Overview**

The **ChatGPT Bulk Delete Tool** is a powerful userscript that allows you to select and delete multiple conversations in ChatGPT at once. It integrates directly with ChatGPT‚Äôs API for reliable and efficient performance, avoiding the issues found in traditional extensions.
